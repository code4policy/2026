DPI-691M: Programming and Data for Policymakers

Course Overview

Data and code are no longer just for programmers. Policymakers from members of congress to analysts and executives need to be equipped with the necessary skills to navigate government in the 21st century. This course is built with the understanding that hands-on experience with programming, data software development, and management methods, open source collaboration, and technology innovation will better prepare students to competently navigate technology projects and nuanced issues at the intersection of technology and governance than a theoretical understanding alone.

Course Philosophy

  • Rolling up their sleeves and writing some code will give students a sense of what all the basic parts of software are and how they fit together.
  • Practical programming and data skills can allow policymakers to be more self-sufficient with regards to basic programming, data collection, and analysis.
  • Participating in the ecosystem of open source software and understanding the spirit of transparency, civic technology, and agile software development methods.
  • Students will become familiar with modern programming technologies and overcome the initial uphill battle that comes before learning to code.

Project Focus

All exercises in the course build towards developing a project that uses the skills learned to deploy a small dynamic web application. Projects are done in small groups, with each group arranged into a "scrum" team. Individual contributions are tracked via GitHub to ensure participation, and groups communicate over Slack.

What We Cover

  • Fundamentals of Unix and the Command Line
  • Github and open source collaboration
  • Web fundamentals (HTTP, HTML/CSS/Javascript)
  • Data collection via APIs and web-scraping
  • Data cleaning and manipulation
  • Visualizing Data for the Web
  • Building and deploying Web Applications
